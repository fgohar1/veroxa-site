# Veroxa Next.js Website

Developer-ready Next.js version of the Veroxa marketing site.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Included pages

- `/`
- `/services`
- `/pricing`
- `/about`
- `/contact`
